# Real-Time Chat Application (Alsancak Chat) 💬

[![Node.js Version](https://img.shields.io/badge/Node.js-18.x-brightgreen)](https://nodejs.org/)

A modern, fast, and responsive real-time chat application built using Node.js, Express, Socket.io, and MongoDB. It provides seamless cross-browser communication with instant messaging features and a live active user tracking system.

## ✨ Features

* **⚡ Instant Real-Time Messaging:** Experience sub-second message delivery powered by WebSockets (Socket.io).
* **👥 Live Active Users List:** See instantly who is online and connected to the chat room.
* **✏️ Dynamic Typing Indicator:** Displays a live notification when another user is actively typing a message.
* **☁️ Persistent Message History:** All chat messages are stored securely in MongoDB, keeping the message history intact even after page refreshes.
* **🔒 Authentication System:** Includes user registration and login systems to keep chats personalized.

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Real-time:** Socket.io
* **Database:** MongoDB (Mongoose)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
